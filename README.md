# Kids

Kids is a Jekyll theme.

## Installation

To use the Kids theme:

1. Add the following to your site's `Gemfile`:

   ```ruby
   gem "kids", git: "https://github.com/dirtyhenry/kids.git", branch: 'master'
   ```

1. Add the following to your site's `_config.yml`:

   ```yml
   theme: kids
   ```
